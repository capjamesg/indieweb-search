IndieWeb Search Advanced Filters

The IndieWeb search engine supports the following advanced filters:

  • coffee OR indieweb - Search for posts that contain the words "coffee" or "indieweb".
  • inurl:"/article/" - Search for posts whose URL contains "/article/".
  • site:"jamesg.blog" - Search for posts on jamesg.blog.
  • discover coffee - Find home pages that mention the term "coffee".
  • xray https://jamesg.blog - Redirect to aaronpk's xray tool to find the microformats markup present on a page.
  • jamesg.blog social - Return all rel=me links on the homepage of a domain.
  • tantek.com inspect feed - Return all feeds associated with a domain home page.
  • random - Go to the homepage of a random site in the search index.
  • category:"coffee" - See posts tagged with the category "coffee".
  • category:"coffee, indieweb" - See posts tagged with either the category "coffee" or "indieweb" (experimental).
  • mf2property:"like" - See posts with a like property in their microformats markup (supports like, rsvp, reply, favorite, share, video, photo, and article) (experimental).

You can use any combination of the filters above.

* This feature is only supported on some pages.
