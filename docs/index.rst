Welcome to IndieWeb Search's documentation!
===========================================

IndieWeb Search is a search engine for websites that adhere to IndieWeb principles and related sites.

IndieWeb Search is maintained as a community resource. Anyone is free to contribute to the search engine.

See the `Contributing <https://github.com/capjamesg/indieweb-search/blob/main/CONTRIBUTING.md>`_ file to learn more about how to contribute to this project.

This wiki contains information about the project, how it works, and how to use the search APIs provided by IndieWeb Search.

The search engine is live at `indieweb-search.jamesg.blog <https://indieweb-search.jamesg.blog>`_.
